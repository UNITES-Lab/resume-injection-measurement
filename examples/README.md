# Examples

This directory contains sanitized synthetic PDF examples based on the case-study
patterns described in the paper. They are not real resumes and do not contain
private applicant data.

## Files

| File | Description |
| --- | --- |
| `pdfs/instruction_override_case.pdf` | Hidden instruction-injection example. |
| `pdfs/data_skills_keywords_case.pdf` | Hidden data-injection example with skill keywords. |
| `results/hcd/` | Full HCD outputs generated by running `rim-hcd` on the example PDFs. |
| `results/vda/` | VDA outputs generated by running `rim-vda` on the example PDFs. |

## Run HCD

```bash
mkdir -p outputs
rim-hcd --pdf examples/pdfs/instruction_override_case.pdf --output outputs
rim-hcd --pdf examples/pdfs/data_skills_keywords_case.pdf --output outputs
```

The end-to-end HCD result files should report `resume_label: 1`. Timing fields
may differ across machines.

## Run VDA

```bash
rim-vda \
  --pdf examples/pdfs/instruction_override_case.pdf \
  --output outputs/instruction_override_case_vda.json
```

This step requires `OPENAI_API_KEY` and Poppler.
